# Curso de aprendizagem de algoritmos aplicados em linguagem Java e PHP. 

Olá sou o professor Cristiano e leceionei programação do período de 2007 a 2021. Neste período muitos alunos me procuravam
com dúvidas em transformar/entender enunciados e algoritmos em um programa em um determinada linguagem de programação.

Repositório dedicado aos vídeos do meu canal do Youtube para aprendizagem de algoritmos em Java e PHP. Os arquivos gerados em cada vídeo serão armazenados em diretórios com o nome Vídeo ###, onde # representa o número do vídeo do canal que será acrescentado o link ao final deste README.

<h1>Objetivo</h1>

<p>O objetivo do curso é ir além do momento sala de aula e aplicar a lógica de algoritmos em uma linguagem de programação diferente 
de Pascal e C com base em um algoritmo estruturado.</p>

<h2>Como o curso será distribuído?</h2>
<ul>
<li>Enunciado do algoritmo e exercício resolvido em português estruturado.</li>

<li>Resolução em JAVA dentro do Eclipse.</li>

<li>Resolução em PHP e HTML no VS Code Studio no servidor XAMPP.</li>
</ul>
